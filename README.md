**3. Цикломатската комплексност**
Цикломатската комплексност на овој код е 9.
Резултатот е добиен според следната пресметка:

ЦК = E - N + 2
E = Number of edges (transfers of control)
N = Number of nodes

=> ЦК = 25 - 18 + 2 = 9
